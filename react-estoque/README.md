# Sistema de Estoque de Tênis

## Funcionalidades
- Login com senha segura (bcrypt)
- Cadastro de usuários
- Controle de estoque (entrada e saída)
- Histórico de movimentações
- Imagens dos produtos

## Tecnologias
- React
- TypeScript
- LocalStorage

## Como rodar o projeto
```bash
npm install
npm run dev
